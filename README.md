# Hospital Management System (DSA)

A Hospital Management System developed in C++ using Data Structures and Algorithms.

## Features
- Add patient records
- Search patient information
- Delete patient records
- Manage hospital data efficiently
- Console-based application

## Technologies Used
- C++
- Code::Blocks
- Data Structures & Algorithms

## Project Structure
- `src/` → source code
- `include/` → header files
- `main.cpp` → entry point
- `Lab_Project.cbp` → Code::Blocks project file
